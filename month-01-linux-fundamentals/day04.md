# Linux Day 04

## Objective

Learned Linux file permissions and how access control works.

## Commands Learned

- ls -l
- chmod

## Concepts

- Read (r)
- Write (w)
- Execute (x)

## Practice

- Viewed file permissions.
- Added execute permission using chmod +x.
- Removed execute permission using chmod -x.
- Modified file access permissions.

## Notes

Linux uses permissions to control access to files and directories. Permissions are divided into owner, group, and others.
